# BRX Bank Simulation

BRX Bank Simulation is a web-based digital banking simulation platform designed to demonstrate modern banking dashboard layouts and user interface flows.

This project is intended for educational, demo, and portfolio showcase purposes.

## Features
- Digital banking dashboard simulation
- Clean and professional UI design
- Responsive layout
- Live cloud deployment
