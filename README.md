# BRX Bank Simulation

BRX Bank Simulation is a web-based digital banking platform prototype that demonstrates the core functionalities and user interface of a modern banking application. The project emphasizes clean design principles, structured user flows, and responsive layouts suitable for desktop and mobile devices.

Users can explore features such as simulated account overview, transaction summaries, and interactive dashboard elements. While the platform does not connect to actual banking systems, it provides a realistic representation of digital banking interfaces for educational and portfolio purposes.

## Features
- Interactive digital banking dashboard simulation
- Clean and modern UI/UX design
- Responsive layout for desktop and mobile devices
- Simulated account overview and transaction summaries
- Designed for educational and portfolio demonstration

## Tech Stack
- **JavaScript / TypeScript** – Primary programming language
- **React** – Component-based frontend library
- **Vite / Next.js** – Modern frontend tooling
- **HTML5 & CSS3** – Structure and styling
- **Tailwind CSS** (optional) – Utility-first CSS framework
- **Vercel** – Cloud deployment and hosting
